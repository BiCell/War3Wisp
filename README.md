# 魔兽改键精灵

面向 **魔兽争霸 3** 的 Windows 改键工具。在后台拦截键盘输入，把你在游戏里实际按下的键，映射成游戏内绑定的快捷键，并支持多套方案一键切换。界面与操作习惯贴近常见「魔兽改键精灵」类产品。

---

## 功能

- **包裹 / 技能改键**：自定义物品栏、技能快捷键映射
- **组合键（一键必杀）**：2 键合一、3 键合一
- **26 套方案**：不同英雄或玩法各存一套，游戏中 `Ctrl + Shift + A~Z` 快速切换
- **仅魔兽前台改键**：默认只在魔兽窗口激活时生效，不影响其他软件
- **显血开关**：友方 / 敌方血条（对应游戏内 HOME / END）
- **禁止 Win 键**：游戏前台时屏蔽误触开始菜单
- **全局热键**：Scroll / Pause 开关改键；方案热键切换
- **配置持久化**：方案保存在用户目录，换目录或更新 exe 不丢配置

## 环境要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 10 / 11 |
| 游戏 | 魔兽争霸 3（含冰封王座等常见版本） |
| 运行方式 | 直接运行发布的 **exe**，**无需安装 Python** |
| 权限 | 首次启动需 **管理员 UAC 授权**（低级键盘钩子需要） |

---

## 快速开始

1. 运行 **魔兽改键精灵.exe**
2. UAC 提示选择 **「是」**
3. 在界面中配置映射，点击 **「保存方案」**
4. 启动魔兽，进入游戏即可使用

> 配置保存在 `%APPDATA%\WarcraftKeyRemapper\config.json`，与 exe 放在哪个文件夹无关。

再次启动已运行的程序时，会提示「已在运行中」，不会打开第二个窗口。

## 使用说明

### 配置映射

1. 顶部选择 **方案 1 ~ 26**
2. 映射表 **左侧** 为游戏内原键，**右侧** 为你要按的物理键
3. 点击输入框后按下目标键（空格显示为 **「空」**）
4. 修改后务必 **「保存方案」**，否则进游戏不生效

### 建议流程

1. 先打开本程序并保存方案，再启动魔兽
2. 保持勾选 **「仅在魔兽争霸窗口前台时改键」**（推荐）
3. 底部 **状态** 显示「运行中」表示当前在改键；窗口最小化后仍生效
4. 点击窗口 **×** 会退出程序并停止改键

### 界面选项

| 选项 | 说明 |
|------|------|
| 显示友方 / 敌方血条 | 与游戏内显血同步（切回魔兽前台时生效） |
| 禁止用 WIN 键 | 魔兽在前台时拦截 Win 键 |
| 启用 / 停用 | 开关改键（同 Scroll / Pause） |
| 方案重置 | 清空当前方案全部映射 |

### 游戏中热键

| 操作 | 按键 |
|------|------|
| 开关改键 | Scroll 或 Pause |
| 切换方案 | Ctrl + Shift + A ~ Z（对应方案 1 ~ 26） |

## 常见问题

| 现象 | 处理 |
|------|------|
| 游戏里没反应 | 确认已 **保存方案**；底部状态为 **运行中**；以管理员运行；魔兽窗口在前台 |
| 其他软件也被改键 | 勾选「仅在魔兽争霸窗口前台时改键」 |
| 换电脑后配置还在吗 | 配置在用户目录 `%APPDATA%\WarcraftKeyRemapper\`，随账户迁移 |
| 能否多开 | 不支持；重复启动会提示已在运行 |
